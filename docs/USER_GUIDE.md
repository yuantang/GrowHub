## 快速开始

### 启动服务

```bash
# 1. 启动后端 API (端口 8080)
cd /path/to/MediaCrawler
uv run python -m uvicorn api.main:app --host 0.0.0.0 --port 8080 --reload

# 2. 启动前端 (端口 5173)
cd web-client
npm run dev
```

### 访问地址

- **前端界面**: http://localhost:5173
- **API 文档**: http://localhost:8080/docs

## 完整使用流程

### 🔑 第一步：准备账号 (账号池)

**位置**: 左侧菜单 → 🛡️ **账号池**

要让爬虫能够抓取数据，您需要提供目标平台的 Cookie。

#### 如何获取 Cookie？

1. 打开浏览器，登录目标平台（如小红书）
2. 按 F12 打开开发者工具
3. 进入 Network（网络）标签页
4. 刷新页面，点击任意一个请求
5. 在 Headers 中找到 `Cookie` 字段，复制全部内容

#### 添加账号

1. 点击 **+ 添加账号**
2. 填写：
   - **账号名称**: 方便识别，如 "小红书-主号"
   - **平台**: 选择 `xhs`（小红书）
   - **Cookie**: 粘贴刚才复制的 Cookie
3. 点击 **保存**
4. 点击该账号的 **"检测健康"** 按钮
5. 确认状态变为 **🟢 Active**

> ⚠️ 提示：Cookie 有效期通常为 7-30 天，过期后需要重新获取。

---

### 📂 第二步：创建监控项目

**位置**: 左侧菜单 → 📂 **监控项目**

这是整个系统的 **核心入口**，一个项目 = 一条完整的自动化流水线。

#### 创建项目

1. 点击 **+ 新建项目**
2. 填写配置：

| 配置项 | 说明 | 示例 |
|-------|------|------|
| **项目名称** | 给这个监控任务起个名字 | "品牌舆情监控" |
| **监控关键词** | 要搜索的关键词，多个用空格或逗号分隔 | "咖啡 探店 测评" |
| **监控平台** | 选择要抓取的平台 | ✅ 小红书 ✅ 抖音 |
| **调度频率** | 多久自动执行一次 | "每 1 小时" |
| **抓取模式** | 搜索方式 | "关键词搜索" |
| **每次抓取数量** | 每次最多抓多少条 | 20 |
| **预警规则** | 发现特定内容时通知 | ✅ 负面内容预警 |

3. **可选**: 勾选 "创建后立即启动自动监控"
4. 点击 **创建项目**

---

### ▶️ 第三步：启动监控

创建项目后，您可以：

| 操作 | 按钮 | 说明 |
|------|------|------|
| **启动** | ▶️ (绿色播放按钮) | 开启定时自动采集 |
| **暂停** | ⏸️ (黄色暂停按钮) | 停止自动采集 |
| **立即执行** | ⚡ (闪电按钮) | 不等倒计时，马上跑一次 |

> 点击启动后，系统会：
> 1. 自动从账号池借一个可用账号
> 2. 按照您设置的频率定时执行爬虫
> 3. 将抓取到的内容存入数据库

---

### 📊 第四步：查看结果

采集完成后，您可以在以下页面查看数据：

| 页面 | 左侧菜单 | 说明 |
|------|----------|------|
| **内容监控** | 🔔 内容监控 | 按时间顺序查看最新抓取的内容 |
| **热点排行** | 🔥 热点排行 | 按热度（点赞+评论+收藏）排序 |
| **数据分析** | 📊 数据分析 | 趋势图表和统计报表 |

---

### ✨ 第五步：使用 AI 创作

**位置**: 左侧菜单 → ✨ **AI 创作**

发现优质内容后，您可以使用 AI 辅助创作：

#### 功能一：智能神评论

1. 复制目标内容的标题或正文
2. 进入 AI 创作页面
3. 选择 "智能神评论"
4. 粘贴内容，选择评论风格（如：专业、幽默、共情）
5. 点击 "生成"
6. 复制生成的评论，发布到目标平台

#### 功能二：爆款改写

1. 粘贴爆款内容
2. 选择 "爆款改写"
3. 选择改写风格
4. 生成后用于自己的内容创作

## 常见问题

### Q: 爬虫抓不到数据？

1. **检查账号状态**: 确认账号池中有 Active 状态的账号
2. **Cookie 是否过期**: 点击 "检测健康"，如果变成 Expired 需要重新获取
3. **关键词是否有效**: 尝试在平台上手动搜索该关键词，确认有结果

### Q: 项目显示"已停止"？

项目创建后默认是停止状态，需要手动点击 ▶️ 启动按钮。

### Q: 如何查看采集进度？

1. 进入 **任务调度** 页面
2. 切换到 **"执行日志"** 标签页
3. 可以看到每次执行的状态、耗时、结果

### Q: 支持哪些平台？

目前支持：
- ✅ 小红书 (xhs)
- ✅ 抖音 (douyin)
- ✅ B站 (bilibili)
- ✅ 微博 (weibo)
- ✅ 知乎 (zhihu)
